# Packet Analysis

During the packet capture, several types of network traffic were observed.

HTTP Traffic

HTTP packets were filtered using the display filter "http". These packets contained requests and responses between the web browser and the web server.

DNS Traffic

DNS queries were filtered using the filter "dns". DNS packets show the process of resolving domain names to IP addresses.

Packet Information

Each captured packet contains several fields including:

Frame number
Source IP address
Destination IP address
Protocol type
Packet length

These fields provide insight into how network devices communicate across the internet.
