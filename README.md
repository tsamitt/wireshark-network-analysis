# Wireshark Packet Analysis 

## Overview

This repository documents a Wireshark focused on analyzing network packets and understanding how common internet protocols operate. The lab involved capturing network traffic, filtering specific protocols, and examining packet contents using Wireshark.

## Objectives

Capture network traffic from a network interface

Identify and analyze HTTP and DNS packets

Understand packet structure and protocol headers

Use Wireshark filters to isolate specific traffic

## Tools Used

Wireshark  
Packet capture (pcap)  
Basic networking protocols  

## Skills Demonstrated

Packet capture and inspection  
Protocol analysis  
Network troubleshooting concepts  
Traffic filtering and investigation
