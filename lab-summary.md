# Lab Summary

In this lab, Wireshark was used to capture and analyze network packets generated during normal web browsing activity.

The packet capture process involved selecting an active network interface card (NIC) and recording traffic during a web download.

The captured packets were then filtered and analyzed to observe common network protocols such as HTTP and DNS.

The goal of the exercise was to gain familiarity with packet analysis tools and understand how network communication occurs between clients and servers.
